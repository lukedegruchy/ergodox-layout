# Extra tricks:

## BKSP hold SHIFT

```
LSFT_T(KC_BSPACE),
```

## ESC hold SHIFT

```
RSFT_T(KC_ESCAPE),
```

## ESC then CTRL

```
LCTL_T(KC_ESCAPE),
```

## TAB then ALT

```
LALT_T(KC_TAB),
```
## APPS then ALT

```
RALT_T(KC_APPLICATION),
```
## CTRL then ENTER

```
RCTL_T(KC_ENTER),
```

## One-tap SHIFT
```
F(F_SFT_L)
```

## One-tap CTRL
```
F(F_CTRL_L)
```

## One-tap ALT
```
F(F_ALT_L)
```

## Combine two keys

```
RGUI(KC_RIGHT)
```
